Scholarship docs guidance

Purpose:
- Keep scholarship-related content in dedicated files named `scholarship.md` or similar.
- Metadata `category=scholarship` and `topic` set to subtopics like `deadlines`, `eligibility`, `application_process`.

Chunking:
- Split per scholarship listing or per deadline/benefit section.
- Avoid embedding curriculum or program lists in scholarship chunks.

Example metadata:
{
  "category": "scholarship",
  "topic": "deadlines",
  "audience": "prospective_student",
  "tags": ["scholarship","deadline"]
}
