Double Degree docs guidance

Purpose:
- Dedicated `double_degree.md` for program-level explanations, partnership lists, eligibility, and application flow.
- Metadata: `category=double_degree`, `topic=international_program` or `topic=partner_university` as appropriate.

Chunking:
- Split into partner-by-partner sections or FAQ-style Q/A per double degree program.
- Do not include full program curriculum or tuition fee details in the same chunk.

Example metadata:
{
  "category": "double_degree",
  "topic": "international_program",
  "audience": "prospective_student",
  "tags": ["double_degree","partnership"]
}
