# Program Recommendation Guidance

## Purpose
Panduan framing untuk merekomendasikan program/jurusan yang sesuai berdasarkan ketertarikan, personality fit, dan kekhawatiran pengguna.

## Key Considerations
- Interest alignment: apa yang user suka lakukan (praktikal vs teoretis)
- Skill match: apakah user nyaman dengan matematika, coding, desain, dsb.
- Career outlook: opsi karir yang realistis terkait program.
- Anxiety/constraint: biaya, waktu, kesiapan akademik.

## Conversational Guidance
- Start with reflective question: "Kamu lebih suka kerja di bidang X atau Y?"
- Provide 2–3 matched program options with short rationale.
- Offer low-cost ways to try: kursus singkat, proyek kecil.

## Recommendation heuristics
- Prefer concrete examples and small experiments.
- Avoid definitive statements; use probabilistic language: "cenderung cocok".
