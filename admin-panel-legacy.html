Kelola Keywords & Auto-Reply

Tambah, edit, atau hapus rule auto-reply berdasarkan keyword

📖 Cara Penggunaan:
  • Keyword: Kata kunci yang akan di-trigger (mis: "halo", "admin", "bantuan")
  • Tipe Cocok:
    • Exact: Cocok jika pesan persis sama dengan keyword
    • Contains: Cocok jika pesan berisi keyword (fleksibel)
    • Regex: Cocok menggunakan regular expression (advanced)
  • Respon: Pesan yang akan dikirim bot ketika keyword cocok
  • Prioritas: Semakin tinggi (0-100), semakin diutamakan jika ada keyword lain yang cocok

💡 Contoh: Keyword="halo", Tipe="contains", Respon="Hai! Ada yang bisa saya bantu?", Prioritas=10

Pengaturan Bot

Atur pesan welcome, threshold waktu, dll

📖 Cara Penggunaan:
  • Nama Setting (Key): Identifier parameter (mis: "welcome_message", "fallback_message", "max_users")
  • Nilai (Value): Isi dari parameter (teks, angka, atau JSON)
  • Penggunaan Umum:
    • welcome_message: Pesan sambutan saat user pertama chat
    • fallback_message: Pesan ketika bot tidak mengerti
    • operating_hours: Jam operasional (JSON: {"start":"09:00","end":"20:00"})
    • max_message_length: Batas karakter respons bot

💡 Contoh: Key="welcome_message", Value="Selamat datang! 👋"

Menu Interaktif

Buat menu numeric (1, 2, 3...) untuk navigasi percakapan

📖 Cara Penggunaan:
  • Key Menu: Identifier unik menu (format: "root.1", "root.2", "root.2.1" untuk submenu)
  • Teks Menu: Deskripsi yang tampil ke user (mis: "Info Produk", "Hubungi Kami")
  • Urutan: Nomor tampilan (user tekan 1, 2, 3 untuk navigasi)
  • Struktur Hirarki:
    • root.1: Menu utama pilihan 1
    • root.1.1: Submenu dari pilihan 1
    • root.2: Menu utama pilihan 2

💡 Contoh: Key="root.1", Teks="📞 Hubungi Kami", Urutan=1 → User tekan 1 untuk pilihan ini

Kirim Broadcast

Kirim pesan ke semua pengguna (opt-in)

📖 Cara Penggunaan:
  • Judul: Ringkasan broadcast (tampil di preview, mis: "Promo Spesial", "Update Sistem")
  • Isi Pesan: Konten lengkap broadcast yang akan dikirim ke user
  • Jadwalkan: Opsi pengiriman:
    • Kosongkan: Kirim sekarang (immediate)
    • Isi tanggal/jam: Kirim pada waktu yang ditentukan (scheduled)
  • Target Pengguna: Otomatis mengirim ke semua user yang terdaftar dan opt-in
  • Tracking: Lihat status pengiriman (queued, sent, failed) di tabel broadcast

💡 Contoh: Judul="Diskon 50%", Isi="Dapatkan diskon 50% untuk semua produk!", Jadwal=[kosong]=kirim sekarang

📖 Cara Membaca Dashboard

Gunakan panel ini untuk memantau kesehatan bot dan perilaku user.

  • Total Users: seluruh user yang pernah chat dengan bot.
  • Active Last Week: user aktif 7 hari terakhir.
  • Handover Rate: persentase chat yang diteruskan ke human agent.
  • Avg Sessions/User: rata-rata jumlah session per user.
  • 📈 Retention: berapa % user yang kembali.
  • 👥 Cohort: analisis kelompok user berdasarkan waktu join.
  • 💬 Popular Topics: topik & keyword yang sering ditanya.
  • 🔥 Heatmap: jam & hari paling ramai.
